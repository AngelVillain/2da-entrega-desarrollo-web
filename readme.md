# Preentrega 2 - CoderHouse / Programación II 🍉

## Descripción del Proyecto
Este sitio web es una propuesta de arquitectura avanzada enfocada en un diseño **Dark Mode** con una paleta vibrante inspirada en el concepto "Watermelon". 

## Tecnologías Utilizadas
* **HTML5:** Estructura semántica para SEO y accesibilidad.
* **CSS3:** Uso de variables nativas (:root), Flexbox y CSS 
* **Bootstrap 5:** Sistema de grillas y componentes responsivos.
* **Git/GitHub:** Control de versiones con commits descriptivos y publicación en GitHub Pages.

## Estructura
- `/css`: Archivos de estilo optimizados.
- `/img`: Recursos visuales.
- `index.html`: Página principal con Hero Section.
- `productos.html`: Listado de servicios con tablas semánticas.
- `contacto.html`: Formulario de contacto integrado.
